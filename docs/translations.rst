Translations
=============

By adding a file called custom_metadata/metadata_fields/translations.json one can add
language specific translations. An example is shown below. The object key is the name of the field,
the value the translated label.


.. code-block:: json
    {
      "de": {
        "age": "Alter",
        "size": "Größe",
        "color": "Farbe"
      },
      "en": {
        "age": "Age",
        "size": "Size",
        "color": "Color"
      }
    }
